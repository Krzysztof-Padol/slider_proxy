## Node Express Mongoose

Example proxy server for flickr API using node + express + mongoose

## Installation and Usage

    $ clone repo
    $ cd node-express-mongoose
    $ npm install
    $ npm start

Project is very simple:
Have just one route, and clue of that project is container under proxy controller
